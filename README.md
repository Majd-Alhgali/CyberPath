# CyberPath

تطبيق تعليمي بستايل Duolingo لتعلم Penetration Testing بشكل تدريجي ومتوافق مع منهج OSCP (PEN-200).

## ⚠️ حالة المشروع الحالية

هذا هو **الهيكل التقني الكامل** للتطبيق (skeleton):
- ✅ كل الشاشات تعمل (الرئيسية، المسار، تفاصيل الموديول، تفاصيل الوحدة، الدرس، الكويز، التقدم، الإعدادات)
- ✅ نظام XP والمستويات (Beginner → Professional)
- ✅ نظام قفل/فتح الوحدات التسلسلي (مثل Duolingo)
- ✅ نظام الـ Streak اليومي والشارات (Badges)
- ✅ التخزين الدائم عبر AsyncStorage (يحفظ تقدمك بين مرات فتح التطبيق)
- ✅ دعم RTL كامل وثيم داكن بألوان terminal (أخضر/سيان)
- ✅ كل 24 موديول من منهج OSCP موجودة بعناوينها ووحداتها (بدون محتوى تعليمي فعلي بعد)

- ⏳ **لا يوجد محتوى تعليمي حقيقي بعد** — كل الوحدات تظهر "المحتوى قيد الإعداد" حتى تتم تعبئتها

## كيف تشغّل المشروع على جهازك (ThinkPad P50 — Ubuntu 24.04)

### المتطلبات
```bash
# تحقق من وجود Node.js (يفضل v18 أو v20)
node -v

# إذا لم يكن مثبتاً، يمكنك استخدام nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
nvm install 20
nvm use 20
```

### تثبيت Expo CLI والاعتماديات
```bash
cd CyberPath
npm install
```

> ملاحظة بخصوص اتصال ADSL البطيء: `npm install` لأول مرة قد يحتاج بعض الوقت (تحميل Expo + React Native + التبعيات ~ 300-400 ميجابايت تقريباً). يفضّل تشغيله أثناء وقت لا تحتاج فيه الإنترنت لشيء آخر.

### تشغيل التطبيق
```bash
npx expo start
```

سيظهر QR Code في الطرفية. لديك خياران:

1. **عبر الهاتف (Redmi 15C):**
   - ثبّت تطبيق **Expo Go** من Google Play
   - تأكد أن الهاتف والـ ThinkPad على نفس شبكة الـ Wi-Fi
   - افتح Expo Go وامسح QR Code

2. **عبر متصفح الويب (للمعاينة السريعة فقط، بعض المكونات قد لا تظهر بدقة كاملة):**
   ```bash
   npx expo start --web
   ```

> إذا كان جهازك والهاتف لا يستطيعان رؤية بعضهما على الشبكة (شائع مع بعض إعدادات الراوتر)، استخدم:
> ```bash
> npx expo start --tunnel
> ```

## كيف تضيف محتوى درس جديد

1. افتح `src/content/lessonTemplate.js` لرؤية الشكل المطلوب بالضبط
2. انسخ القالب وعبّئ الحقول لدرسك الجديد
3. أضف الكائن إلى المصفوفة المناسبة في `src/content/module_XX/index.js`
   (مثال: `lessonsUnit02.push({...})` لدرس في موديول 5، الوحدة 2)
4. اربط المصفوفة بالـ `units` في `src/content/curriculum.js` وغيّر `isContentReady` إلى `true`

## بنية المشروع
```
CyberPath/
├── App.js                      # نقطة الدخول، تفعيل RTL
├── src/
│   ├── content/
│   │   ├── curriculum.js       # خريطة المنهج الكاملة (24 موديول)
│   │   ├── lessonTemplate.js   # قالب الدرس الفاضي
│   │   └── module_05/          # مثال على مجلد محتوى موديول
│   ├── screens/                # كل شاشات التطبيق
│   ├── components/              # المكونات القابلة لإعادة الاستخدام
│   ├── store/                  # Zustand stores (التقدم، الإعدادات)
│   ├── navigation/              # Bottom Tabs + Stack Navigators
│   └── theme/                  # الألوان والخطوط
```
